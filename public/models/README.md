Place `face-api.js` model files in this folder.

Required (for the current prototype):

- `tiny_face_detector_model-weights_manifest.json` + corresponding shard files
- `face_landmark_68_model-weights_manifest.json` + corresponding shard files
- `face_recognition_model-weights_manifest.json` + corresponding shard files

This app loads models from `/models` at runtime.
